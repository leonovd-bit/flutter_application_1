## Purpose

## Navigation Role
- **From**: WelcomePageV3 or LoginPageV3
- **Next**: EmailVerificationPageV3 (email signup) or DeliverySchedulePageV3 (social signup)

## Key Features
- Email/password registration
- Input validation
- Terms acceptance
- Social sign-up options
- Form error handling
- Account creation with Firebase Auth
